# Gridle

Play the Wordle game for energy nerds: https://gridle.bendnorman.com/!

# Development requirements

To test and develop new features, make sure you meet the following requirements:

| **item** | **version** |
| -------- | ----------- |
| node.js  | >=14        |
| npm      | >=8         |

# Run a local copy

```
npm ci
npm start
```
